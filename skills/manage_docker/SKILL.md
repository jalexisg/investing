# Manage Docker

> **Description**: Utilities for managing the Docker environment of the Investing application.

## Instructions
This skill provides context for managing the docker container.

### Common Commands
- **Build and Start**: `docker-compose up --build`
- **Stop**: `docker-compose down`
- **View Logs**: `docker-compose logs -f`
- **Shell Access**: `docker-compose exec web bash` (assuming 'web' is the service name)

## Configuration
The `Dockerfile` and `docker-compose.yml` are located in the root directory.
